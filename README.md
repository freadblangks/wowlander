## Introduction

BfaCore is mostly based on TrinityCore, written in c++ and sql.



## Requirements

* Visual Studio 2019
* OpenSSL 1.1.1h
* Boost 1.66
* MySql 5.7.32



Software requirements are available in the TrinityCore [wiki](https://www.trinitycore.info/display/tc/Requirements) for
Windows, Linux and macOS.
